Make a resource localizable

You can make any type of resource localizable, including images and audio files. For example, provide an image that is more culturally sensitive for a particular language or region.

  1. In the Project navigator, select the resource.

  2. In the inspector, under Localization, click Localize.

  3. In the dialog that appears, choose the development language for the resource (for example, Base) from the pop-up menu, then click Localize.

  4. In the inspector, under Localization, select the languages that you want the resource to be localized in.

In the project navigator, the resource becomes a group containing language-specific files that show the language in parenthesis. (In the file system, Xcode creates separate language folders, named [language ID].lproj, to store the language-specific resources.)
