Use code completion to enter symbols

As you write code, the source editor analyzes your text and the surrounding context and displays a list of suggested symbols. Select one to insert it or continue typing to refine the list of suggestions.

Important: Code completion is fully functional only after Xcode has indexed your project.

Insert a suggested symbol

  1. In the source editor, begin typing the name of a symbol.

    A list of suggestions for the symbol appear. If all of the possible completions have a common string, the string is highlighted. If there is no common string, code completion shows parts of the string highlighted.

  2. Select an item in the suggestion list.

  3. Press Return to accept the suggestion.

Dismiss the suggested symbol list

  • Press the Esc (Escape) key or click elsewhere in your code.

View a description of the suggestion

  • Click a suggested symbol to view its description below the suggestion list.

Jump between parameters or arguments in an inserted method or function

When a method or function contains parameters or arguments, code completion includes a placeholder for each.

  • To move to the next placeholder, choose Navigate > Jump to Next Placeholder (Tab).

  • To move to the previous placeholder, choose Navigate > Jump to Previous Placeholder (Shift-Tab).

Turn off code completion

  • In Text Editing preferences, click Editing and deselect the option “Suggest completions while typing.”

    You can still invoke code completion manually by pressing Control–Space.
