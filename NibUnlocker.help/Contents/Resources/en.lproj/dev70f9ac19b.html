asset types

Type

Content

App icon

Image files for the various sizes of an icon for an app or Messages extension. The set includes a large icon image for the App Store.

Apple TV image stack

A set of layered images combined to enable parallax.

Apple TV image stack layer

An image for a layer in an image stack.

Apple TV Top Shelf image

An image for the App that is displayed in the Top Shelf on Apple TV. The Top Shelf image is part of a tvOS App icon set.

Color set

A named color that is device-specific or the same on all devices.

Cube texture set

The images for a 3D texture mapped onto a cube.

Data set

Any binary data file except executable code.

Game Center dashboard image

An image for the logo in a Game Center dashboard.

Game Center leaderboard

An Image stack for the background of a Game Center leaderboard.

Game Center leaderboard set

An image stack for the background of a Game Center leaderboard set.

Generic macOS icon

Image files for the various size of a macOS version in the older .iconset format.

Image set

The different resolutions of an image used for UIImage, NSImage, user interface controls, and other assets using images.

Launch image

An app launch image.

Sprite atlas

The image sets for an SKTextureAtlas class.

Sticker pack

A pack of stickers for the Messaging app.

Sticker

An image for a sticker in a sticker pack.

Sticker sequence

An animated sticker.

Sticker frame

An image for a frame in a sticker sequence.

Texture set

An image for a 2D texture.

Watch complication

An image set for each supported watch complication type.

For information on types and sizes, see the watchOS human interface guidelines.
