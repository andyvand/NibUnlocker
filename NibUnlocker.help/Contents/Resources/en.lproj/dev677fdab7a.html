Add headers to a target

You can explicitly associate public, private, or project header files with a target using the Headers build phase.

  1. In the project editor, choose the target from the Project/Targets pop-up menu, then click Build Phases.

  2. If a Headers build phase doesnâ€™t appear, click the Add button (+), then choose New Headers Phase.

  3. Next to Headers, click the disclosure triangle icon to reveal the files.

  4. Under Headers, click the Add button (+), choose the header in the sheet, then click Add.
