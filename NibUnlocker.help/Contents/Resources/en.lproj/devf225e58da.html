Create a simulator configuration

If you don’t see the device type with the OS version you want to test in Simulator, you can create a custom configuration using the Devices and Simulators window. The new simulator will appear in the run destination of the Scheme menu in the toolbar.

  1. Open the Devices and Simulators window (choose Window > Devices and Simulators), then click Simulators.

  2. In the lower-left corner, click the Add button (+).

  3. In the sheet that appears, enter a simulator name, choose a device type, and choose an OS version.

  4. If you don’t see the OS version you want to use, choose “Download more simulator runtimes.”

    The Components preferences opens where you can download additional simulator OS versions.

  5. Click Create.
