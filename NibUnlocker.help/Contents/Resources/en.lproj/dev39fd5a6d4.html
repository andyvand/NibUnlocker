locked file

To protect files from being accidentally changed or deleted, you can lock them in Finder: Select the file, choose File > Get Info and select Locked. When a file is locked, you can move it to the Trash, but you can’t empty the Trash with the document in there. To edit the file, you must unlock it. Xcode can be set to automatically unlock files: Open Xcode > Preferences > General and select the “Automatically unlock files” checkbox.
