Simulate a runtime location in the scheme editor

You can simulate a location for your app when it runs in Xcode using the Run scheme action.

  1. In the main window, choose Edit Scheme from the scheme menu in the toolbar.

  2. In the left column, select the Run scheme action, then click Options.

  3. If necessary, select the Allow Location Simulation checkbox under Core Location.

  4. Choose a location from the Default Location pop-up menu:

    • None: Donâ€™t simulate a location.

    • [City, Country]: Simulate the location of the city.

    • Add GPX File to Project: Use a GPS Exchange Format file that simulates the location data.

      If you choose this option, choose a GPX file and click Add in the dialog that appears. The GPX file may have waypoints, tracks, and routes that can be used to simulate one or more static locations, or location changes over time.

  5. Click Close.
