device configuration pane

In Interface Builder, use the device configuration pane to configure the current device used to lay out objects on the canvas, and to create variations of the user interface for different configurations. A device configuration is a set of properties—for example, device family, orientation, adaptation, and interface style—that affects the layout, appearance, and capabilities of an app.

To open the device configuration pane, click the “View as” button in the layout bar.
