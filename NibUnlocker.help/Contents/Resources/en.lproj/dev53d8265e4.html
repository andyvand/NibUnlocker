Add, move, and rename playground pages

Create separate lessons by adding pages to your playground. Then you can rename the playground pages and change the order.

Add a new page

  1. Open the Project navigator (choose View > Navigators > Show Project Navigator).

  2. Choose File > New > Playground Page.

    The first time you add a page to a playground, two new pages appear in the playground group. One contains the current content of the playground editor and the other is the new page.

Rename a page

  1. In the Project navigator, select the page and press Return.

    The name of the page becomes editable.

  2. Enter the new name of the playground page into the text field and press Return.

Change the order of pages

  • In the Project navigator, drag a page to a new location in the list of pages.
