View and manage the constraints for an item

Use the Size inspector () to view and manage all of the constraints for an item.

View all of the constraints for an item

  1. On the Interface Builder canvas or in the outline view, select the desired item.

  2. Open the Size inspector.

  3. Scroll the inspector pane until you see the list of constraints.

Filter the list of constraints for an item

  1. On the Interface Builder canvas or in the outline view, select the desired item.

  2. Open the Size inspector.

  3. Scroll the inspector until you see the filter panel above the list of constraints.

    The filter panel shows all of the possible attributes for a constraint. Each blue line indicates an attribute of a constraint shown in the list.

    The screenshot shows two filter panels. The panel on the left is for a view that does not include a baseline constraint, such as a UIView. The panel on the right is for a view that does contain baseline constraint, such as a UILabel.

  4. On the filter panel at the top of the list, select a blue line to show only those constraints that include the selected attribute.

    The screenshot shows a filter panel with the baseline constraint selected. The list shows only the four constraints that include a baseline attribute.

Delete a constraint for an item

  • In the Size inspector, select one or more constraints and press the Delete key.
