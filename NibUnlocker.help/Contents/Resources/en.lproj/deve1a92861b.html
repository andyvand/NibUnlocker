Refactor your code

Use the Editor > Refactor submenu and the Action menu to refactor code.

What kind of refactoring can I do?

  • Rename a symbol in a file or in your project.

  • Extract code to a function or method.

  • Extract code to a variable.

  • Add a method or property to a class or protocol.

  • Add a parameter to a function or method.

  • Add missing protocol requirements or function definitions.

  • Add an elements to an existing code statement such as adding an else clause to an if statement.

  • Add a localization wrapper for a string.

When do I use the Action menu?

Use the Action menu to see a context sensitive list refactoring choices for the selected code structure.

When do I use the Editor > Refactor submenu?

Use the menu to refactor code based on the current selection or the location of the insertion point.
