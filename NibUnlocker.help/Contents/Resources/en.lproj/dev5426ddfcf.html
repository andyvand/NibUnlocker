Add, delete, rename, and share schemes

Use the scheme management dialog to manage the schemes in your project. Choose Manage Schemes from the scheme menu.

Automatically create schemes for your targets

  • To automatically create new schemes whenever you add new targets, select the “Autocreate schemes” checkbox. To create schemes for any targets that don’t already have schemes, click the Autocreate Schemes Now button.

Create a new scheme

  1. Click the Add button (+).

  2. Choose a target from the Target pop-up menu.

  3. Enter a name for the scheme.

  4. Click OK.

Delete a scheme

  1. Select a scheme in the list.

  2. Click the Delete button (–).

  3. Click the Delete button in the confirmation dialog.

Edit a scheme

  1. Select a scheme in the list.

  2. Click the Edit button to display the scheme editor.

Hide or show a scheme

  • To hide a scheme from the scheme menu, deselect the checkbox in the Show column for the scheme. To show a hidden scheme again, select the checkbox in the Show column for the scheme.

Duplicate a scheme

  1. Select a scheme in the list.

  2. Click the Action icon () at the bottom of the list and choose Duplicate from the pop-up menu.

    The scheme is duplicated and the scheme editor opens.

  3. Configure the new scheme.

  4. Click Close to accept your configuration and dismiss the scheme editor.

Rename a scheme

  1. Select a scheme in the list.

  2. Click the name of the scheme in the Scheme column.

  3. Enter a new name.

Reorder schemes in the scheme menu

  • Select a scheme in the list and drag it up or down to the desired location.

Move a scheme to a different project or workspace

  • Select the project or workspace from the pop-up menu in the Containers column for the scheme.

Export a scheme

You can export a scheme to

  1. Select a scheme in the list.

  2. Click the Action icon () at the bottom of the list and choose Export from the pop-up menu.

  3. Select an output folder on your computer.

  4. Click Export.

Import a scheme

  1. Click the Action icon () at the bottom of the list and choose Import from the pop-up menu.

  2. Locate a scheme file on your computer.

  3. Click Import.

Share a scheme

  • To share a scheme with Xcode Server or others on your team, select the checkbox in the Shared column for the scheme. To stop sharing a scheme, deselect the checkbox in the Shared column for the scheme.
