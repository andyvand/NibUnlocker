Start or stop Xcode Server

After you set up Xcode Server, you can start and stop the server in Server & Bots preferences.

Start the server

  1. Choose Xcode > Preferences, then click Servers & Bots.

  2. In the upper-right corner, click the switch to turn it from OFF to ON.

  3. To view the status of the system account that runs Xcode Server, click Settings.

  4. If the system account appears logged out, click Log In, enter the password, then switch back to the account running Xcode.

    A green icon should now appear next to the system account indicating that it is logged in and running the server.

Stop the server

  1. Choose Xcode > Preferences, then click Servers & Bots.

  2. In the upper-right corner, click the switch to turn it from ON to OFF.
