View symbol declarations and reference documentation

You can view declarations and reference documentation for symbols directly in the source editor and the Quick Help inspector.

View the declaration for a symbol

  • To navigate to the declaration, Control-Command-click the symbol in the source editor. If the definition is in a separate file, the source editor opens that file. Alternatively, you can click the symbol and choose Navigate > Jump to Definition.

  • To open the assistant editor and display the declaration, Option-Command-click the symbol in the source editor. This approach lets you keep the symbol in view as you inspect its definition. Alternatively, you can click the symbol, hold down Option, and choose Navigate > Jump to Definition in Assistant Editor.

View reference documentation for a symbol in the source editor

  • To display a Quick Help popover describing the symbol, Option-click the symbol.

  • To dismiss the Quick Help popover, click anywhere in the main window.

Tip: To view more extensive documentation, click links in the Quick Help popover.

View reference documentation for a symbol in the inspector

  1. To display the Quick Help inspector, open the inspector area () and click the Quick Help button ().

  2. To display documentation for the symbol in the Quick Help inspector, click a symbol in the source editor.

Tip: You can click links in the Quick Help inspector to view more extensive documentation.
