modal segue attributes

The screenshot shows the modal segue Attributes inspector.

Attribute (Screenshot letter)

Description

Identifier (A)

Enter a string used to identify a segue at runtime.

The attribute corresponds to the identifier property of UIStoryboardSegue.

Class (B)

Set a custom class for the runtime segue object.

Module (C)

Select the module containing the implementation of the segue class.

Kind (D)

Set or change the type of segue.

Changing the type can change the attributes shown in the inspector.

Presentation (E)

Choose how the modal view controller is displayed onscreen from the pop-up menu.

The attribute corresponds to the modalPresentationStyle property of UIViewController. The other choices in the pop-up correspond to the values of UIModalPresentationStyle.

Transition (F)

Choose the transition style used for presenting the modal dialog from the pop-up menu.

The attribute corresponds to the modalTransitionStyle property of UIViewController. The other choices in the pop-up correspond to the values of UIModalTransitionStyle.

Animates (G)

Select the checkbox to enable a transition animation at runtime.

Peek & Pop (H)

Select the checkbox to add peek and pop segues for 3D Touch.
