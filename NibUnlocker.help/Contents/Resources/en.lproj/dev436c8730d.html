Add an existing remote to your local repository

If both the local and remote repositories exist, you can add the existing remote repository to your local repository.

  1. In the Source Control navigator (), Control-click the working copy, then choose Add Existing Remote from the pop-up menu.

  2. In the sheet that appears, enter the repository name and location.

  3. Click Add.
