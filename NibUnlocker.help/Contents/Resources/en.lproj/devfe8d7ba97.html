Edit a constraint

Edit the most common properties for a constraint in a pop-up editor, or edit all the properties using the Attributes inspector.

Edit a constraint using the pop-up editor

  1. On the Interface Builder canvas or in the outline view, double-click on the desired constraint.

  2. Set the values in the pop-up editor.

Edit a constraint in the Attributes inspector

  1. On the Interface Builder canvas or in the outline view, select the desired constraint.

  2. In the inspector area, select the Attributes inspector.

  3. Set the desired values in the Attributes inspector.
