Specify runtime arguments and environment variables for your app

Use the scheme editor to specify runtime arguments and environment variables for your app to use when it runs in Xcode.

  1. In the main window, choose Edit Scheme from the scheme menu in the toolbar.

  2. In the left column, select the Run scheme action, then click Arguments.

  3. To add a runtime argument, click the Add button (+) at the bottom of the Arguments Passed On Launch table and enter the argument.

  4. To add an environment variable, click the Add button (+) at the bottom of the Environment Variables table and enter the variable name and value.

  5. Click Close.
