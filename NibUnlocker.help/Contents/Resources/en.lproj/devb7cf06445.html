Monitor the status of builds

Check the activity view and build reports to monitor the status of builds.

View the status of the current build

  • Check the activity view at the top of the main window.

The activity view shows the active progression of the current product build, and reports on success and failure.

View detailed build history

A build report displays a list of tasks performed by the build system, as well as warnings and errors encountered.

  1. Go to the Report navigator.

  2. Filter the reports, if desired. See View and filter logs and reports.

  3. Click a build report.

    The selected build report is displayed in the editor area.

  4. Xcode performs builds incrementallyâ€”build results are cached whenever possible and rebuilt only when necessary. Click All to see the complete list of build tasks performed, including cached ones. Click Recent to see only incremental build tasks performed.

Save a detailed build history transcript

  1. Select a build report in Report navigator. See View detailed build history.

  2. Click in the Save button in the editor area.

  3. When prompted, choose an output location for the transcript.

  4. Open the saved transcript in the text editor of your choice, such as TextEdit.

    Note: You can also extract detailed build report entries by selecting them in the editor area, copying them to the clipboard, and pasting them into a text editor, such as TextEdit.
