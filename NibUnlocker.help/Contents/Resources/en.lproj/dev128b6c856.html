Verify, reveal, and delete installed toolchains

Verify, reveal, or delete installed toolchains in the Toolchains pane of Components preferences. This pane is visible only when an alternative toolchain is installed.

Xcode supports using alternative toolchains, such as toolchains downloaded from Swift.org, instead of the default tools for indexing, building, and debugging your projects. Alternative toolchains must be manually downloaded and installed into /Library/Developer/Toolchains/ or ~/Library/Developer/Toolchains/ to be recognized by Xcode and appear in the Toolchains pane of Components preferences.

Verify the code signature of an installed toolchain: Hover over an installed toolchain, click the Action button (), then choose Verify Code Signature from the pop-up menu.

Reveal an installed toolchain in the Finder: Hover over an installed toolchain, click the Action button (), then choose Show in Finder from the pop-up menu.

Delete an installed toolchain: Hover over an installed toolchain, click the Action button (), then choose Move to Trash from the pop-up menu.

Alternatively, Control-click an installed toolchain to display the Action menu.
