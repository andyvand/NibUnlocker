About Core Data mapping model editor

Use the Core Data mapping model editor to change a Core Data migration mapping model (a file with a .xcmappingmodel file extension). To open the Core Data mapping model editor, select a Core Data mapping model file in the Project navigator and the file opens in the editor area.

The outline view shows the entity mappings in the mapping model. These can be selected individually for viewing or editing.

The detail area shows details about the selected entity mapping in the outline view.

The mapping inspector shows details about the selected entity, attribute or relationship mapping.

Add entity mapping: At the bottom of the Core Data mapping model editor, click the Add Entity Mapping button.

Set models for mapping: At the bottom of the Core Data mapping model editor, select the source and destination models from the pop-up menus.
