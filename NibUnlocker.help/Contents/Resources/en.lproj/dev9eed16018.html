Add a particle emitter to your project

With the SpriteKit particle emitter editor, you can experiment with SpriteKit particle effects and see the results immediately. The visual interface separates the task of designing particle effects from the programming so that artists can create effects independent of your code.

The editor lets you modify many of the properties of a particle emitter, including:

  • Where the particles are created

  • The number of particles created

  • The rotation, size, and movement of the particles

  • How the particle changes throughout its lifetime

Note: Before adding a particle emitter to your project, you must have a SpriteKit-enabled project. Create a new project from a Game project template or link a target in an existing project to the SpriteKit framework.

Add a particle emitter to your project

  1. Choose File > New > File.

  2. Choose Resource > SpriteKit Particle File.

  3. Select one of the preinstalled particle emitter textures, and click Next.

  4. Enter a name for the emitter file in the Save As field.

  5. Select the checkbox associated with your project and click Create.

    Xcode creates a particle emitter file with the extension .sks.

  6. Select the new particle emitter file in the Project navigator.

    Xcode opens the file in the particle emitter editor.
