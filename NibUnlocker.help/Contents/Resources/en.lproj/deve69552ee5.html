Edit deployment info settings

The default deployment settings are sufficient for development, but it’s best to review these settings before you distribute your app. Some settings must match values you enter in App Store Connect later.

Set the deployment target

The deployment target specifies the lowest operating system version that can run your app. For example, the lowest available setting for iPad apps is iOS 4.3.

  1. In the project editor, choose the target and click General.

  2. If necessary, click the disclosure triangle next to Deployment Info to reveal the settings.

  3. Choose the OS version you want to target from the Deployment Target pop-up menu.

Set the target devices (iOS, watchOS)

  1. In the General pane of the project editor, click the disclosure triangle next to Deployment Info.

  2. From the Devices pop-up menu, choose iPhone, iPad, or Universal (to target both types of devices).

Set device orientations and other UI options (iOS, watchOS)

  1. In the General pane of the project editor, click the disclosure triangle next to Deployment Info.

  2. In the Device Orientation section, select the device orientations your app supports.

  3. In the Status Bar Style section, choose a light or dark style from the pop-up menu.

    Choose Default (dark style) for light backgrounds and Light for dark backgrounds.

  4. To hide the status bar, select the “Hide status bar” checkbox.

  5. If your app uses the full screen, select the “Requires full screen” checkbox.

Set the user interface style (tvOS)

  1. In the General pane of the project editor, click the disclosure triangle next to Deployment Info.

  2. In the User Interface Style section, choose a style from the pop-up menu.

    Choose Automatic, Dark, or Light.
