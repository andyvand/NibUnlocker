Learn about a build setting

Use the Quick Help inspector to see details for a selected build setting in Xcode, or view the reference documentation for build settings in Xcode Help.

View build setting details in Xcode. Select a build setting in the Build Settings pane of the project editor to display its declaration, value type, and description in the Quick Help inspector.

View build setting reference documentation in Xcode Help. Go to Build settings reference. Use Find (Command-F) to quickly find and jump to a specific setting.
