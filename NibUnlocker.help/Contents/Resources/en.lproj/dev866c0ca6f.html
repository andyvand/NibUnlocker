popover segue attributes

The screenshot shows the popover segue Attributes inspector.

Attribute (Screenshot letter)

Description

Identifier (A)

Enter a string used to identify a segue at runtime.

The attribute corresponds to the identifier property of UIStoryboardSegue.

Class (B)

Set a custom class for the runtime segue object.

Module (C)

Select the module containing the implementation of the segue class.

Kind (D)

Set or change the type of segue.

Changing the type can change the attributes shown in the inspector.

Directions (E)

Select the preferred arrow directions for the popover.

The attribute corresponds to the permittedArrowDirections property of the UIPopoverPresentationController.

Anchor (F)

Drag from the circle in the field to the view to use for the anchor rectangle of the popover.

The attribute corresponds to the sourceView property of the UIPopoverPresentationController.

Passthrough (G)

Drag from the circle to each of the views a user can interact with while the popover is active. There can be multiple passthrough views.

The attribute corresponds to the passthroughViews property of the UIPopoverPresentationController.

Animates (H)

Select the checkbox to enable a transition animation at runtime.

Peek & Pop (I)

Select the checkbox to add peek and pop segues for 3D Touch.
