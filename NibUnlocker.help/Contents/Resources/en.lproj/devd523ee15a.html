code editing options

The following code editing options are configurable in Xcode > Preferences > Text Editing > Editing:

Option

Description

Line numbers

Show line numbers in the gutter of the source editor.

Cold folding ribbon

Control that appears in the source editor to identify, fold, and unfold structures.

Page guide at column

Show a vertical page width guide at a specified position in the source editor.

Highlight instances of selected symbol

Highlight all instances of the selected symbol in the current source file after a specified delay.

Editor overscroll

Sets the amount of scroll past the end of the file in the source editor. The possible values are:

  • None

  • Small

  • Medium

  • Large

Suggest completions while typing

Automatically display code completion suggestions as you enter code in the source editor.

Use Escape key to show completion suggestions

Display code completion suggestions when pressing the Escape key. If this setting is deselected, the Escape key instead dismisses the scope bar and other accessory views in the source editor.

Automatically insert closing braces ("}")

Automatically insert a closing brace ("}") when typing an opening brace ("{").

Enable type-over completions

Refine the list of code completion suggestions as you continue typing.

Automatically balance brackets in Objective-C method calls

Automatically insert an opening bracket when typing an unbalanced closing bracket in an Objective-C method call.

Enclose selection in matching delimiters

Enclose the selected text in matching delimiters by typing an open delimiter such as an opening brace ("{").

Automatically trim trailing whitespace

Automatically trim white space from the ends of lines.

Including whitespace-only lines

Automatically trim white space from empty lines.

Default text encoding

Set the character set that the source editor uses to display and save files.

Default line endings

Set the default line ending style for new files. You may need to change this setting if you use other development tools that require source files with specific line endings.

Convert existing files on save

Convert existing files to use the default line ending styles when saved.

Show iteration counts

Show code coverage iteration counts.
