Set the placeholder intrinsic size for a view

Use a placeholder intrinsic size to indicate a view’s width and height and avoid design time constraint ambiguity.

  1. On the Interface Builder canvas or in the outline view, select the desired view.

    The view must be the target of at least one constraint.

  2. In the inspector area, select the Size inspector.

  3. Choose Placeholder from the Intrinsic Size pop-up menu.

  4. Set appropriate values for the width and height.

    Select None if a value is not required.

Depending on the position, size, and orientation of the item or items that you Control–drag to, the shortcut menu displays positioning and size attributes. The attributes fall into three categories:

Horizontal positioning: Leading, Trailing, and Center. (To choose from Left and Right instead of Leading and Trailing, use the Attributes inspector to deselect “Respect language direction” in the item’s pop-up menu.

Vertical positioning: Top, Bottom, Center, and Baseline. (The Baseline attribute is available only when you select two items, and both have baselines. Views rendering text are the most common items with baselines.)

Size: Width, Height, and Aspect Ratio.
