Fold or collapse areas of your code

To better focus on code that you are currently editing, you can fold or collapse other areas of your code in the source editor. You can fold and unfold code by using the Action menu, the code folding ribbon, or the Editor menu.

Fold code using the Action menu

  1. Hold the Command key and move the pointer over the structure you want to fold.

  2. Command-click the highlighted structure, then choose Fold from the Action menu.

    The folded code is represented by an ellipsis (…).

If you don’t enable the code folding ribbon, double-click the ellipsis to unfold the code.

Enable the code folding ribbon

Use the code folding ribbon to more quickly identify, fold, and unfold structures.

  1. Choose Xcode > Preferences, then click Text Editing.

  2. Click Editing, then select the “Code folding ribbon" checkbox.

Fold and unfold code using the ribbon

When the code folding ribbon is enabled, a grey ribbon appears in the gutter between the line numbers and the structures or comments you can fold.

  1. Hover the pointer over the ribbon next to the structure.

  2. Click anywhere in the control that appears in the ribbon.

    • If the structure is unfolded, it is folded. The folded code is represented by an ellipsis (…).

    • If the structure is folded, it is unfolded. Alternatively, double-click the ellipsis to unfold the code.

Fold code using the Editor menu

Use the Editor > Code Folding submenu to fold and unfold the code structure at the insertion point, all methods and functions in the current file, or all code blocks in the current file.

Menu item

Description

Fold

Fold the structure surrounding the insertion point.

Unfold

Unfold the folded code structure at the insertion point.

Unfold All

Unfold all the folded code structures in the current file.

Fold Methods & Functions

Fold each of the methods and functions in the current file.

Unfold Methods & Functions

Unfold any folded methods and functions in the current file.

Fold Comment Blocks

Fold each of the comment blocks in the current file.

Unfold Comment Blocks

Unfold any comment blocks in the current file.
