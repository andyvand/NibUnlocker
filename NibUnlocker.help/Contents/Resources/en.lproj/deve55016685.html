SpriteKit particle emitter editor

With the SpriteKit particle emitter editor, you can experiment with particle effects and see the results immediately. The visual interface separates the task of designing particle effects from the programming so that artists can create effects independent of your code.

The editor lets you modify many of the properties of a particle emitter, including:

  • Where the particles are created

  • The number of particles created

  • The rotation, size, and movement of the particles

  • How the particle changes throughout its lifetime

Note: Before adding a particle emitter to your project, you must have a SpriteKit-enabled project. Create a new project from a Game project template or link a target in an existing project to the SpriteKit framework.
