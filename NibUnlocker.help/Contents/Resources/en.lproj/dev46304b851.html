Place views using guides

Add custom guides to any view controller on the canvas for precise view placement. For Apple TV apps, add standard column size guides.

Enable snap to guides

  • Open a storyboard or Xib file in Interface Builder and check Editor > Snap to Guides.

Add a horizontal or vertical guide to a view

  1. On the Interface Builder canvas or in the outline view, select the desired view or view controller.

    The selected view must either be:

    • A subview in the hierarchy below the main view of a view controller.

    • A view that is not inside a view controller.

  2. Choose Editor > Guides > Add Vertical Guide or Add Horizontal Guide.

    A yellow guide line appears on the canvas.

  3. Drag the guideline to the required position.

    As you drag the guideline, numbers on each side of the guide show the distance from the edge. The distance is based on the coordinates of the view the guide was added to.

    On a vertical guide the numbers appear on the top at left and right of the line. The left number shows the distance from the left edge of the enclosing top level view. The right number shows the distance from the right edge.

    On a horizontal guide the numbers appear on the left at top and bottom of the line. The top number shows the distance from the top edge of the enclosing top level view. The bottom number shows the distance from the bottom edge.

Remove a horizontal or vertical guide

  • On the Interface Builder canvas, drag the guide out of the view or view controller.

Add column guides for Apple TV grids

  1. On the Interface Builder canvas or in the outline view, select the desired view controller.

  2. Choose a column layout from the Editor > Guides submenu.
