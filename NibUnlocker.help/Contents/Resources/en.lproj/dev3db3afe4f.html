Rename a project or an app

Use the File inspector to give a project or an app a new name.

  1. Select your project in the Project navigator.

  2. In the Identity and Type section of the File inspector, enter a new name into the Name field.

  3. Press Return.

    A dialog is displayed, listing the items in your project that can be renamed. The dialog includes a preview of how the items will appear after the change.

  4. To selectively rename items, disable the checkboxes for any items you donâ€™t want to rename. To rename only your app, leave the app selected and deselect all other items.

  5. Click Rename.
