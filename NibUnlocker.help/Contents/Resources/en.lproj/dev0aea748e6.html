user defined runtime attribute

A user defined runtime attribute is a custom runtime attribute for an object that does not have a corresponding Interface Builder inspector. When the nib file is loaded, the custom attribute you specify is initialized with a call to setValue:forKeyPath:.

You add, edit, and delete an attribute using the User Defined Runtime Attributes editor in the Identity inspector.

Key Path: The path for the attribute in the instantiated object.

Type: The type for the attribute.

Value: The value for the attribute.

Important: The property or key path for the user defined runtime attribute must exist in the object otherwise an exception will occur.
