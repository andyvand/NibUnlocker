Export localizations for translation

You can export all the files that need to be localized from your Xcode project, or export the files for specific languages.

  1. In the Project navigator, choose the project, then choose Editor > Export For Localization.

  2. In the export dialog that appears, enter a folder name, select a location, select the languages you want to include (under Localizations), then click Save.

Xcode creates a folder containing an Xcode Localization Catalog (xcloc) (a folder with a .xcloc file extension) for each language you select. The XML Localization Interchange File Format (XLIFF) files in the catalog contain the localizable strings that Xcode extracts from:

  • The arguments to the NSLocalizedString or similar macro in your code.

  • User interface strings files.

  • Other strings and stringsdict files in the project.

    Important: Xcode adds the correct plural variants for each language to the stringsdict files.

  • Siri intent definition files.
