View and adjust performance tests

Navigate your performance tests in the Test navigator. View the results of performance test results and adjust their baselines in the editor area.

View performance test results

  1. Run a performance test method.

  2. Select a performance test method in the Test navigator to examine it in the editor area.

  3. Click the performance test result in the editor area or click its icon in the gutter to open the performance result popover. A graph shows the performance of the last ten runs relative to one another. Click a number beneath the graph to see the performance of that run.

Set a performance test baseline or standard deviation allowance

  1. Run a performance test method.

  2. Select a performance test method in the Test navigator to examine it in the editor area.

  3. Click the performance test result in the editor area or click its icon in the gutter to open the performance result popover.

  4. To use the performance from the most recent run as the new baseline, click the Accept button. To manually set a baseline, enter a value into the Baseline text field.

  5. To change the standard deviation allowance, enter a percentage into the Max STDDEV text field.

  6. Click the Save button.
