Add, edit, and view rich comments

To add formatted text to your playground, use the playground markup format. The markup format supports headings, rulers, lists, code voice, emphasis, bold, links, assets, and more. You can view the rendered text by switching to rendered markup mode and edit the markup by switching to raw text mode.

The markup below renders as a title followed by a bulleted list.

/*:### Some Animals* Cat* Dog* Llama*/

View the rendered markup: Choose Editor > Show Rendered Markup.

Edit the raw text: Choose Editor > Show Raw Markup.

For more information on using markup, see Markup Formatting Reference.
