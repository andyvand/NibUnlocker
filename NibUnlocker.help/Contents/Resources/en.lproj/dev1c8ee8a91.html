Add media objects to the user interface

You can enhance the user experience by adding media elements such as icons, graphics, and audio to the user interface from the Media library. The Media library contains media resources that you previously added to your project.

  1. In Interface Builder, click and hold the Object library button () in the toolbar, then choose Show Media Library from the pop-up menu.

    The Media library opens in a separate window. Alternatively, Option-click the button to open the Media library in a persistent window.

  2. To filter the media, enter text in the filter bar at the top of the library.

  3. To toggle between grid and list view, click the button in the upper-right corner of the library.

  4. Drag the media you want to add from the library to the canvas.

    In the canvas, Interface Builder highlights a valid destination in blue and displays an Add icon.
