Add an AR resource group, image, and object

You can add AR image and object resources to the app bundle. First create an AR resource group, then add an AR image or object asset to the group. To learn more about AR assets, go to ARReferenceImage and ARReferenceObject.

  1. In the Project navigator, select the asset catalog.

  2. Click the Add button (+) at the bottom of the outline and choose "New AR Resource Group" from the pop-up menu.

    The new folder appears in the outline view.

  3. To add an AR image to the selected folder, choose “New AR Reference Image” from the Add button (+) pop-up menu.

  4. To add an AR object to the selected folder, choose “New AR Reference Object” from the Add button (+) pop-up menu.

  5. Drag the AR files from the Finder to the appropriate wells in the detail area.

  6. In the Attributes inspector, optionally, enter a name for the image or object.

    The name you enter will appear as the AR image or object name property at runtime.
