Add, edit, and delete a user defined runtime attribute

Add a user defined runtime attribute that is initialized when the nib file is loaded.

Add an attribute

  1. On the Interface Builder canvas or in the outline view, select the object.

  2. Choose View > Utilities > Show Identity Inspector.

  3. In the User Defined Runtime Attributes editor of the Identity inspector, click the plus (+) button to add an attribute.

  4. In the Key Path column in the editor for the new attribute, enter the key path in the object for the new attribute.

  5. In Type column of the for the new attribute, click on the current type to select a new type from the pop-up menu.

  6. In the Value column for the new attribute, enter a default value for the attribute.

Edit an attribute

  1. On the Interface Builder canvas or in the outline view, select the object.

  2. Choose View > Utilities > Show Identity Inspector.

  3. In the User Defined Runtime Attributes editor of the Identify inspector, edit the key path, type, and value of the desired attribute.

Delete an attribute

  1. On the Interface Builder canvas or in the outline view, select the object.

  2. Choose View > Utilities > Show Identity Inspector.

  3. In the User Defined Runtime Attributes editor of the Identify inspector, select the desired attribute.

  4. In the the User Defined Runtime Attributes editor editor, click the minus (-) button to delete the selected attribute.
