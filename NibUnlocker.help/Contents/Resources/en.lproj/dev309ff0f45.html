certificate name

The certificate name for a signing certificate appears in Keychain Access. Use the format of the name to identify the certificate type and purpose.

Name

Description

iPhone Developer: Person’s Name (Member ID)

An iOS Development certificate used to run an iOS, tvOS, or watchOS app on devices and use certain app services during development.

iPhone Distribution: Team Name (Team ID)

An iOS Distribution certificate used to distribute your iOS, tvOS, or watchOS app on designated devices for testing or to submit it to the App Store.

Mac Developer: Person’s Name (Member ID)

A Mac Development certificate used to enable certain app services for a macOS app during development and testing.

3rd Party Mac Developer Application: Team Name (Team ID)

A Mac App Distribution certificate used to sign a macOS app before submitting it to the Mac App Store.

3rd Party Mac Developer Installer: Team Name (Team ID)

A Mac Installer Distribution certificate used to sign and submit a Mac Installer Package, containing your signed app, to the Mac App Store.

Developer ID Application: Team Name (Team ID)

A Developer ID Application certificate used to sign a macOS app before distributing it outside the Mac App Store.

Developer ID Installer: Team Name (Team ID)

A Developer ID Installer certificate used to sign and distribute a Mac Installer Package, containing your signed app, outside the Mac App Store.
