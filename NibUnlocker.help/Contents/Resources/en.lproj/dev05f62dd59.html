Switch toolchains

Switch between toolchains in the Toolchains pane of Components preferences. This pane is visible only when an alternative toolchain is installed.

Xcode supports using alternative toolchains instead of the default tools, such as toolchains downloaded from Swift.org, for indexing, building, and debugging your projects. Alternative toolchains must be manually installed into /Library/Developer/Toolchains/ or ~/Library/Developer/Toolchains/ to be recognized by Xcode and appear in the Toolchains pane of Components preferences.

If you use an alternative toolchain, you might encounter regressions because the alternative tools won’t have been fully qualified by Apple for official release.

Switch to a different toolchain

  • Select the radio button beside a toolchain to activate it.

    You can activate an alternative toolchain or reactivate the default Xcode toolchain.

An activated toolchain remains active across Xcode launches for all of your projects until you switch toolchains again. When an alternative toolchain is active, the toolbar of the project window displays an alternative toolchain indicator () in the activity view. Click this indicator to open the Toolchains pane of Components preferences.

You can also switch toolchains by choosing Xcode > Toolchains > «Toolchain Name» in the menu bar.
