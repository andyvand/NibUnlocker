Configure Siri (iOS, watchOS)

To handle user requests that originate from Siri, enable Siri on the iOS app target, then add an intents extension target to your project.

To learn more about intents extensions, go to Creating an Intents App Extension.

Enable Siri

  1. In the project editor, select the iOS app target, then click Capabilities.

  2. In the Siri section, click the switch to turn it from OFF to ON.

    The Siri entitlement is added to the entitlements file (a file with a .entitlements file extension) and App ID.

Add an intents extension target

  1. Choose File > New > Target.

  2. Select either the iOS or watchOS platform.

  3. Under Application Extension, select Intents Extension, then click Next.

  4. Enter the name of the extension and optionally, change the other options.

  5. For iOS apps, if you want to customize portions of the Siri interface, select the Include UI Extension checkbox.

  6. Click Finish.
