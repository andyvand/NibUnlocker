peek and pop segue attributes

The screenshot shows Peek & Pop section the Attributes inspector.

Attribute (Screenshot letter)

Description

Preview (A)

Choose Custom from the pop-up menu to add a different identifier for the peek segue.

Identifier (B)

Enter a string used to identify the peek segue at runtime.

This field shows when Preview is set to Custom.

Commit (C)

Choose Custom from the pop-up menu to change the kind of the pop segue.

Identifier (D)

Enter a string used to identify the pop segue at runtime.

This field shows when Commit is set to Custom.

Kind (E)

Set or change the type of the pop segue.

Changing the type can change the attributes shown in Peek & Pop section of the inspector.
