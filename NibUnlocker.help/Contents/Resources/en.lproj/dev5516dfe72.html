Add resources to a playground

If you want to access resources such as images, data, or sounds in the playground code, add them to the Resources folder.

  1. Open the Project navigator (choose View > Navigators > Show Project Navigator).

  2. In the Project navigator, select the Resources folder for the playground or a page, click the Add button (+) in the filter bar, then choose Add Files to "Resources” from the pop-up menu (or choose File > New > Add Files to "Resources”).

  3. In the dialog that appears, select the resource file, then click Add.

    The resource file appears in the Resources folder.

In your playground or page, access resources from your Swift code or in markup delimiters using the name of the file. You do not need to include Resources/ at the beginning of the name.

For example, the following code initializes an NSImage with the file called Image.png:

let theImage = NSImage(named: "Image.png")
