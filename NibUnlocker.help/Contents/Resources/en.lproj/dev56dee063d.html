language ID

A language ID identifies a language, dialect, or script.

For a language used in many regions, use a code that represents a language designator. For example, to specify English, use the en language designator. Use the two-letter ISO 639-1 standard (preferred) or, if no code is available for a particular language, use the ISO 639-2 standard instead.

To distinguish between different languages and regional dialects, use a language designator with a region designator and a script designator separated by hyphens. For example, to specify the English language as it is used in the United Kingdom, use en-GB. To represent Mandarin Chinese, spoken in Taiwan, written in Traditional Chinese script, use zh-Hant-TW.

To specify a script, combine a language designator with a script designator separated by a hyphen, as in az-Arab for Azerbaijani in the Arabic script.

fr-CA for French in Canada
