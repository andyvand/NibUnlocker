Compare revisions of a file

Use the version editor to compare revisions of your files that you commit to a source code repository, view the authors who committed changes, and display the log messages.

Compare revisions

  1. In the Project navigator, click the Source Control icon in the filter bar.

    The modified files appear in the Project navigator.

  2. Select a file in the navigator area.

    The file opens in the editor area.

  3. In the toolbar, choose Comparison () from the version editor pop-up menu.

    In the editor area, the local revision of the file appears on the left and the last committed revision appears on the right. For each change, the comparison editor shows the line that changed and highlights the changes to the line.

  4. Use the controls below the editor area and between the revisions to go to the next and previous change.

  5. Optionally, choose other revisions to compare from the jump bars below the editor area.

    Choose a different revision from the jump bar under the right and the left panes. In the jump bar menu, there’s a menu item for each revision of the file. The revisions appear in chronologically order from the current local revision to the oldest committed revision.

Discard individual changes

  1. In the comparison editor, click a change in the gutter between the two revisions and choose Discard Change.

  2. In the sheet that appears, click Revert.

    If you don’t want to see this sheet again, click “Do not show this message again” and then click Revert.

Find out who made the changes

  • From the version editor pop-up menu, choose Authors ().

    In the editor area, the persons who made the changes to the revision appear to the right of the file.

View the source control log

  • From the version editor pop-up menu, choose Log ().

    In the editor area, the log entry for the revision appears to the right of the file.
