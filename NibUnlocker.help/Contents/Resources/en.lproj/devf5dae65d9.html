show, show detail, and custom segue attributes

Show, show detail, and custom segues have the same attributes. The screenshot shows the show segue Attributes inspector.

Attribute (Screenshot letter)

Description

Identifier (A)

Enter a string used to identify a segue at runtime.

The attribute corresponds to the identifier property of UIStoryboardSegue.

Class (B)

Set a custom class for the runtime segue object.

Module (C)

Select the module containing the implementation of the segue class.

Kind (D)

Set or change the type of segue.

Changing the type can change the attributes shown in the inspector.

Animates (E)

Select the checkbox to enable a transition animation at runtime.

Peek & Pop (F)

Select the checkbox to add peek and pop segues for 3D Touch.
