Add, remove, and edit Cocoa bindings

Use Cocoa bindings to synchronize the value of a user interface element attribute with the value of an application model object property.

Add a Cocoa binding by control-dragging

  1. In the assistant editor, open the source file containing the desired property.

  2. On Interface Builder canvas, control-drag from an interface element to a property in the source file.

    Valid properties highlight as you drag over them.

  3. Configure the binding in the Cocoa binding popover.

Remove a Cocoa binding

  1. On the Interface Builder canvas or in the outline view, select the desired interface element.

  2. Choose View > Utilities > Show Bindings Inspector.

  3. In the Bindings inspector, show the settings for the bound attribute.

  4. In the Bindings inspector, deselect the “Bind to” checkbox.

Edit a Cocoa binding

  1. On the Interface Builder canvas or in the outline view, select the desired interface element.

  2. Choose View > Utilities > Show Bindings Inspector.

  3. In the Bindings inspector, show the settings for the bound attribute.

  4. In the Bindings inspector, edit the desired settings.
