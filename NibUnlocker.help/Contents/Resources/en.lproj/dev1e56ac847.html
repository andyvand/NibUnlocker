Snippets library

Use the Snippets library to add code snippets (reusable code fragments) to your source files.

Open the Snippets library: Select a source file in the project navigator, then click the Snippets library button () in the toolbar or choose View > Libraries > Show Library.

Add a snippet to your source file: Drag the snippet from the library to the source file.

Filter the snippets shown in the library: Type text into the filter bar.

Open the Snippets library in a persistent window: Option-click the Object library button () in the toolbar.
