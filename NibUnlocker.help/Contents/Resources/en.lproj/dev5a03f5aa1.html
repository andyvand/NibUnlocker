Xcode Localization Catalog (xcloc) format

Xcode Localization Catalog is a folder with a .xcloc file extension that Xcode uses to export and import localization files. The Xcode Localization Catalog supports all localized assets, not just the strings files, and provides additional contextual information for localizers.

The Xcode Localization Catalog folder contains the following file and folders:

File

Description

contents.json

JSON file containing metadata about the localization catalog, such as the development region, the target locale, the tool (for example, Xcode) and its version number, and the catalog version number.

Localized Contents

Localizable resources, including the language-specific XML Localization Interchange File Format (XLIFF) files ([language ID].xliff) and folders for language-specific resources (language ID].lproj). Resources may include image files, audio files, and user interface files.

Source Contents

Assets used to produce the content that provides context for localizers, such as user interface files and other resources.

Notes

Additional information for localizers, such as screenshots, movies, or text files.

Note: Xcode Localization Catalog is supported in Xcode 10 and later.
