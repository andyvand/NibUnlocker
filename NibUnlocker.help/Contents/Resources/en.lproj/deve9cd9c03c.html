Add code using the Action menu

Add a template such as a method, case statement, or parameter to a code structure.

  • Command-click on the structure, then choose a code template from the Action menu.

Menu item

Description

Add Property

Add a property to a class, protocol, or struct with placeholders for the name and type.

Add Method

Adds a method to a class, protocol, or struct with a placeholder for the name.

Add Parameter

Add a parameter to a function or method with placeholders for the name and type.

Add Return Type

Add a type for the return value of a function or method with a placeholder for the type.

Add “else” Statement

Add an else clause to an if statement with a placeholder for the code.

Add “else if” Statement

Add an else if clause to an if statement with placeholders for the condition and code.

Add “case” Statement

Add a case to a switch statement statement with a placeholder for the code.

Add “default” Statement

Add the default case to a switch statement with a placeholder for the code.
