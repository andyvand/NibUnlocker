Debug an app or process that is already running

Start a debugging session with an app that is already running on the Mac, including apps running in Simulator.

Debug a process as the logged in user

  1. If necessary, open the project for the running app.

  2. To start the session, choose Debug > Attach to Process, then selecting the app or process from the submenu.

Debug a process as root or with a different toolchain

  1. If necessary, open the project for the running app.

  2. Choose Debug > Attach to Process by PID or Name.

  3. In the sheet that appears, enter the name of the process ID (PID) or app name. Optionally, choose the PID or name from the menu that appears as you type.

  4. From the Toolchain pop-up menu, choose the desired toolchain.

  5. Select root as the user.

    This option is available only if you have permission to debug the process as root.

  6. Click Attach.
