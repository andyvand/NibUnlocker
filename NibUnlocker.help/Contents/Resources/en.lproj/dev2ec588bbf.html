Set the copyright key (macOS)

Make sure that your information property list (Info.plist) contains a valid value for the Copyright key. For details on possible values, see NSHumanReadableCopyright in Information Property List Key Reference.

  1. In the Project navigator (), choose the target, then click Info.

  2. In the target properties table, double-click in the Value column of the Copyright (human-readable) row.

  3. Enter a new value for the key.
