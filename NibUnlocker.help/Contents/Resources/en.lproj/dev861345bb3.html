Extract code to new functions and variables

You can create a new function or variable from existing code. The selected code is moved to a new function and replaced by a call to the new function. When you extract code to an expression, the selected code is set to the value of a new variable, and replaced by the new variable. If the selected code contains multiple occurrences of the same expression, that expression is set to the value of a new variable, and each occurrence is replaced with the new variable.

  1. Command-click the code to show the Action menu.

  2. Choose Extract Function, Extract Expression, or Extract Repeated Expression.

    Extract Function: Creates a function with the contents of the code.

    Extract Variable: Creates a variable set to the contents of the code.

    Extract All Occurrences: Creates a variable set to a repeated expression in the contents of the code.

    After completing the extraction, the name of the new function or variable is selected.

  3. Enter a new name for the function or variable, then press Return.
