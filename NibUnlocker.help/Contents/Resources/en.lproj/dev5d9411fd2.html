Add an interactive live view

You can use live views to add interactivity to your playgrounds, experiment with different user interface elements, and build your own custom elements. Add an interactive live view to a playground by importing PlaygroundSupport and setting the live view of the current page to your custom view or view controller.

Note: If you select the Single View template when you create a playground, your code already sets the live view. Just open the assistant editor and run the code.

Add a live view

  1. In your code, import PlaygroundSupport.

    Add this line of code:

    import PlaygroundSupport

  2. Set the live view for the page to an instance of your view or view controller.

    For example, the Single View template sets the live view to an instance of MyViewController:

    PlaygroundPage.current.liveView = MyViewController()

  3. Open the assistant editor in the playground (choose View > Assistant Editor > Show Assistant Editor), then run the code.

    The live view renders in the assistant editor and will continue running until you stop it or an error occurs.
