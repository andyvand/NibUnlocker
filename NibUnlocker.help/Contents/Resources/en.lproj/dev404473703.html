Configure game controllers (tvOS)

If your app uses a game controller, you can add the Game Controller framework to your project and specify the types of controllers your app supports.

  1. In the project editor, choose a target, then click Capabilities.

  2. In the Game Controllers section, click the switch to turn it from OFF to ON.

    The Game Controller framework is added to your project.

  3. Optionally, specify the supported types of game controllers.

    Click the checkbox next to the type of controller your app supports.

    • Extended Gamepad: An extended gamepad to be used with the GCExtendedGamepad class. To require a game controller for your app, only select this option.

    • Micro Gamepad: A micro gamepad to be used with the GCMicroGamepad class.

    See GCSupportedGameControllers for details.

  4. In the table, you can drag a controller type to reorder it in the list.
