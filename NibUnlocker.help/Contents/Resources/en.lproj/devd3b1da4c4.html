Add a test class to a project

Add a test class to expand the scope of testing in a project with new methods.

  1. Click the Add button (+) at the bottom of the Test navigator.

  2. Choose New UI Test Class or New Unit Test Class from the pop-up menu.

  3. Enter a name for the class in the Class field.

  4. Choose a superclass from the “Subclass of” pop-up menu. By default, classes are subclasses of XCTestCase.

  5. Choose an implementation language from the Language pop-up menu.

  6. Click the Next button.

  7. Select a destination, group, and test target.

  8. Click the Create button.

    A new class is added to the target, appears in the Test navigator, and opens in the editor area. The new class includes setUp, tearDown, testExample, and testPerformanceExample template methods.

  9. Expand the template methods and write new test methods.
