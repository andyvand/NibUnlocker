Manage fonts and colors

Use Fonts & Colors preferences to customize the appearance of source code and console text. You could, for example, set Xcode to display code comments in a large, purple font.

A theme defines a set of font and color configurations for use in a specific environments. Xcode includes a number of preconfigured themes, including: Default, which works well for most people; Dusk, which applies a dark background and bright colors for use at night; and Presentation Large, which is intended for projecting highly visible code during presentations. You can add, remove, and customize themes.

Activate a font and color theme

  • Select a theme in the left column.

Edit a font and color theme

  1. In the left column, select a theme.

  2. Near the top of the pane, click Source Editor or Console.

  3. Select one or more syntax categories.

  4. To change the font or text size of the selected syntax categories, click the Fonts window button, then make your selections in the Fonts window.

  5. To change the text color of the selected syntax categories, click the font color button, and choose a new color in the Colors window.

  6. To change the spacing, choose an option from the spacing pop-up menu.

  7. To change the cursor style, choose a style from the cursor pop-up menu.

  8. To change the color of the background, selection, cursor, or other attribute of the source editor or console, click a color button at the bottom of the pane, and choose a new color in the Colors window.

Add a custom font and color theme

  1. At the bottom of the left column, click the Add button (+).

  2. Select one of the default themes to serve as a base from the menu, or select Duplicate “[Theme Name]” to duplicate the currently selected theme.

  3. Update the font, text size, and colors of the new theme.

Rename a theme

  1. In the left column, select a theme.

  2. Click the theme’s name to enter edit mode.

  3. Enter a name for the theme.

  4. Press Return.

Remove a font and color theme

  1. In the left column, select a theme.

  2. At the bottom of the left column, click the Delete button (–).

Restore a default font and color theme

  1. At the bottom of the left column, click the Add button (+).

  2. Select a default theme from the menu to restore it.
