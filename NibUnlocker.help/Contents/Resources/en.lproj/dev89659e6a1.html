Create asset variations for different device traits (iOS, tvOS, watchOS)

In Interface Builder, you can add, remove, and edit variations of an asset for different device traits.

Varying traits in the Attributes inspector

  1. Select an asset catalog to display its contents.

  2. In the set list, select the desired asset.

  3. In the inspector area, select the Attributes inspector.

  4. In the Attributes Inspector, add, remove, and edit the desired trait settings.

Varying an image for light and dark style (tvOS)

  1. On the Interface Builder canvas, select the image view.

  2. In the Attributes inspector, click the plus (+) button next to image name or to the highlighted image name.

  3. In the pop-up menu that appears, select the desired variation.

  4. Set the image name for the new variation choosing an image from the pop-up menu or by entering the name of the image.
