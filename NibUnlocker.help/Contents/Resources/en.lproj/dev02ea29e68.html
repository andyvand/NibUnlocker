Configure how files are opened and displayed

Use Navigation preferences to select the editor area that displays files when they’re opened by clicking, Option-clicking, and double-clicking. Files can be opened in the standard editor, an assistant editor, a new window, or a new tab. A summary of the current behavior appears at the bottom of the window.

Activate a new tab or window when it opens

  • Select the “When a tab or window opens, make it active” checkbox.

Set where a file opens when you select it

  • Select one of the following navigation options:

    Uses Primary Editor: Files open in the standard editor of the current window and tab.

    Uses Focused Editor: Files open in the currently active editor—the standard editor or an assistant editor—of the current window and tab.

Set where to open a file when you Option-click it

  • From the “Optional Navigation” pop-up menu, choose one of the following options:

    Uses Single Assistant Editor: Files open in the assistant editor. Files that are Option-clicked in the assistant editor, such as from the jump bar, open in the standard editor.

    Uses Separate Assistant Editor: Each file opens in a new assistant editor in the current window and tab. If the file is already open in an editor, that editor becomes active.

    Uses Separate Tab: Each file opens in a new tab. If the file is already open in a tab, that tab becomes active.

    Uses Separate Window: Each file opens in a new window. If the file is already open in a window, that window becomes active.

Set where to open a file when you double-click it

  • From the “Double Click Navigation” pop-up menu, choose one of the following options:

    Uses Separate Tab: Each file opens in a new tab. If the file is already open in a tab, that tab becomes active.

    Uses Separate Window: Each file opens in a new window. If the file is already open in a window, that window becomes active.

    Same as Click: Files open in the standard editor of the current window and tab.
