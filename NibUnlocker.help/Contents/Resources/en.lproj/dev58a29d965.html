blend modes

The Blend Mode pop-up menu provides several different options corresponding to the SKEmitterNode particleBlendMode property.

Mode

Description

Alpha

Creates transparent particles that are blended with the background image.

Add

Adds the pixel values of the particle and underlying images. Creates a white pixel if this value is greater than 1.

Subtract

Subtracts the particle pixel value from the underlying pixel values. Creates a black pixel if this value is less than 0.

Multiply

Creates a darker particle effect by multiplying the pixel values of the particle with the underlying pixel values.

MultiplyX2

Creates a dark particle effect by multiplying the pixel values of the particle with the underlying pixel values and then doubling the result.

Screen

Creates a lighter particle effect by inverting the pixel values, multiplying them together, and then inverting them again.

Replace

Uses the particle’s color only; it does not blend with background pixels.
