Add a README or other Markdown file

Add a README or other Markdown file to your project and edit it using the source editor.

  1. Choose File > New > File.

  2. Under Other (scroll to the bottom of the sheet), choose Markdown File, then click Next.

  3. Enter a location and file name (README.md), then click Create.

    The file opens in the editor area of the main window.

Tip: To go to a section in the Markdown file, choose the heading from the jump bar in the source editor.
