Adjust keyboard shortcut mappings for menus and text operations

Use Key Bindings preferences to view and customize keyboard shortcuts within Xcode. Each key binding associates an action, such as selecting a menu or initiating an operation on selected text, with a sequence of keys to serve as a shortcut for initiating that action. You can use the default set of key bindings, or create your own sets and switch to them at any time.

View specific actions or keyboard shortcuts

  • Browse through the list of actions and keyboard shortcuts.

    • To filter the list of actions by type, click an action category, such as All, Menu, or Text, above the list of actions. The Menu category represents Xcode menu actions. The Text category represents text-editing operations.

    • To filter for a specific action or search term, enter the term into the search field.

      In the search field, click the search options icon () and choose whether you want to search action descriptions, keyboard shortcuts, or both.

Change the keyboard shortcut for a menu or text action

  1. Locate the action for which you want to change the shortcut.

  2. Double-click the shortcut to select it.

  3. Type a new shortcut to replace the selected one.

  4. Click elsewhere to apply your change.

Remove a keyboard shortcut for a menu or text action

  1. Locate the action for which you want to remove the shortcut.

  2. Double-click the shortcut to select it.

  3. Click the Delete button (–) to the right of the shortcut. (Don’t press the keyboard Delete key; that just sets the action’s shortcut to Delete.)

  4. Click elsewhere to apply your change.

Add an additional keyboard shortcut to a text action

Text actions can optionally be associated with multiple keyboard shortcuts.

  1. Locate the text action for which you want to add an additional shortcut.

  2. Double-click the shortcut to select it.

  3. Click the Add (+) button and type the new shortcut.

  4. Click elsewhere to apply your change.

Locate and resolve conflicting keyboard shortcuts

Each shortcut can be associated with a single action. If multiple actions have been assigned the same shortcut, there is a conflict.

  1. Above the list of actions, click Conflicts.

  2. Adjust the shortcuts to eliminate the conflicts

Note: Some system-level shortcuts may conflict with Xcode shortcuts and take precedence.

Create a new set of keyboard shortcuts

  1. Choose Manage Key Bindings from the Key Bindings Set pop-up menu.

  2. In the lower left corner of the dialog, click the Add button (+).

  3. Choose Duplicate [SelectedSetName] or a template set from the pop-up menu.

  4. Enter a name for the new key bindings set.

  5. Click Done.

Switch between sets of keyboard shortcuts

  • From the Key Bindings Set pop-up menu, choose a key binding set.

Tip: The Keyboard Viewer is a convenient visual aid for interpreting shortcut key sequences. To add the Keyboard Viewer to your menu bar, open the Keyboard preference pane in System Preferences, click the Input Sources tab, and select the “Show Input menu in menu bar” checkbox. Then choose Show Keyboard Viewer from the input menu in the menu bar.
