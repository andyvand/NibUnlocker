Add placeholders for missing protocol properties and methods

When you add a protocol or when you get an error that your code is not conforming to a protocol, you can add placeholders for all the missing properties and methods using the Action menu.

  1. Command-click on the class or struct to show the Action menu.

  2. Choose Add Missing Protocol Requirements.

    Templates for any missing properties, methods, or other elements are inserted at the top of the class or struct.
