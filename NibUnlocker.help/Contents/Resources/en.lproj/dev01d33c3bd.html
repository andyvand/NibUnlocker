See the value of a variable at runtime

You can see the current value of a variable at runtime by setting a breakpoint and hovering over the variable in the source editor when execution pauses.

  1. Add a breakpoint to the line containing the variable you want to inspect.

  2. Run your app.

  3. When execution pauses, place the pointer over the variable in the source editor.

    A popover appears, showing the current value of the variable.

  4. Expand the disclosure triangle in the popover to see more detailed information about the variable.

Tip: You can also view the values of variables in the variables pane of the debug area.
