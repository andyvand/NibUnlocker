Open a file containing a symbol

  1. Choose File > Open Quickly (Command-Shift-O) from anywhere in your project.

  2. In the search field popover, enter some or all of the symbol name.

  3. Select an item in the list of completions and press Return to open the file.

    Use the up and down arrow keys to navigate the list of completions.
