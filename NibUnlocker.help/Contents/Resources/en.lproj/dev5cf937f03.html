Build variations for different platforms

You can conditionalize build setting to produce product variations for different architectures and SDKs.

  1. In the project editor, choose the project or target, click Build Settings, then locate the build setting you want to edit.

    To quickly find a build setting, search for the build setting or filter build settings.

  2. Click the disclosure triangle to the left of the build setting to display all available build configurations.

  3. Hover the pointer over a build configuration and click the Add button (+).

  4. Choose an architecture and SDK from the conditional pop-up button.

  5. Choose or enter a value for the conditionalized build setting. See Configure build settings.
