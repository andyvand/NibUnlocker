Set a simulated screen size

Use a simulated size to change the size of a view controller on the Interface Builder canvas.

  1. On the Interface Builder canvas or in the outline view, select the desired view controller.

  2. Choose View > Utilities > Show Attributes Inspector.

    The Attributes inspector for the selected controller appears in the inspector area. The inspector includes the Simulated Metrics area with choices for the Status, Top, and Bottom Bars as shown below:

  3. Click the size pop-up menu.

    A pop-up menu opens showing the possible configurations:

  4. Choose the item that best matches your desired configuration.

    Inferred: Sets the size depending on the type of controller and any relationship connections. This is the default setting.

    Freeform: Use the Size inspector to set the size.

    Page Sheet: Set the size to a page sheet modal presentation.

    Form Sheet: Set the size to a form sheet modal presentation.

    Master: Set the size to a master view controller.

    Detail: Set the size to a detail view controller

Changing simulated size of a view controller changes the size for child view controllers that have a simulated size of “Inferred”.
