Align objects using the alignment rectangle

View objects using the alignment rectangle when aligning objects on the Interface Builder canvas. The alignment rectangle includes any runtime adornments—such as bevels or shadows—added by the system.

Switch between frame and alignment rectangles

  1. Select the view in the canvas or outline view.

  2. Choose View > Utilities > Show Size Inspector.

    The Size inspector for the selected view opens in the inspector area.

  3. In the View section, choose either Frame Rectangle or Alignment Rectangle from the Show pop-up menu.

    The frame rectangle is the raw frame of the view element. The alignment rectangle is the total visual space occupied by the view element including any adornments. The alignment rectangle may be the same as the frame rectangle.

Show all alignment rectangles

  • Choose Editor > Canvas > Show Layout Rectangles.

    The alignment rectangle for each object appears in blue on the canvas.

Show all frame rectangles

  • Choose Editor > Canvas > Show Bounds Rectangles

    The frame rectangle for each object appears in blue on the canvas.
