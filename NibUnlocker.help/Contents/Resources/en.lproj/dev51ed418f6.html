Cocoa binding popover

Use the Cocoa binding popover to configure the binding between interface element attribute and a model object property.

Connection: The type of connection. This is always “Binding” for a Cocoa binding.

Interface object: The interface object.

Bind: The attribute of the interface object to bind to the model object.

Controller: The controller containing a reference to the model object.

Custom class: The class of the controller.

Key path: The path in the controller object to the model object.

Model object: The model object class.

Key path: The path to the property in the model object to bind to the interface object attribute.
