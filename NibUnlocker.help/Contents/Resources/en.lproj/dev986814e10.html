Configure object attributes

Use the Attributes inspector to configure the selected object.

  1. On the Interface Builder canvas or in the outline view, select the object.

    Objects include view controllers, views, and any other item created from an item in the Object library.

  2. Choose View > Utilities > Show Attributes Inspector.

  3. Use the Attributes inspector to configure the desired attributes.

    The changes you make are applied to the object immediately.
