Create and use code snippets

A code snippet is a reusable code fragment that you can insert in a source file by dragging it from the Snippets library () or by entering a keyword in the source editor. You can use system snippets or create your own.

Drag a code snippet from the Snippets library

  1. In the source editor, click the Snippets library button () in the toolbar.

    The library opens in a separate window. Alternatively, Option-click the button to open the Snippets library in a persistent window.

  2. To filter the snippets, enter text in the filter bar at the top of the library.

  3. To see the code for a snippet, select it in the list.

    A dialog appears containing the code. Click Done to dismiss it.

  4. Drag a snippet from the library to a location in the source file.

  5. In the source editor, edit the snippet.

    If the snippet contains placeholders, edit the highlighted placeholder, then press Tab to move to the next placeholder.

Insert a code snippet using a keyword

  1. In the source editor, enter the keyword for the snippet (for example, the name of an expression).

  2. In the dialog that appears, select the snippet (if necessary), then press Return.

  3. In the source editor, edit the snippet.

    If the snippet contains placeholders, edit the highlighted placeholder, then press Tab to move to the next placeholder.

Tip: The code completion dialog includes snippets for common expressions such as switch and while statements. Just begin entering the expression, then select the snippet from the dialog that appears.

Create a custom code snippet

  1. In the source editor, select the code fragment you want to make into a snippet, then choose Editor > Create Code Snippet.

    Alternative, just choose Editor > Create Code Snippet.

  2. In the dialog that appears, edit the properties of the snippet.

    • Title: Enter the name of the snippet displayed in the Snippets library.

    • Summary: Enter the description of the snippet displayed in the Snippets library.

    • Platform: Choose the platform for the snippet from the pop-up menu.

    • Language: Choose the programming or script language from the pop-up menu.

    • Completion Shortcut: Enter a keyword for the snippet that can be used in the source editor.

    • Completion Scope: Choose the scope for the snippet from the pop-up menu. Possible values are:

      • All: Can be inserted anywhere.

      • String or Comment: Can be inserted where string or comments are allowed.

      • Top Level: Forthcoming

  3. In the text field, edit the code for the snippet.

    To include tokens—such as placeholders for parameter values of a method—in a snippet, insert a string using the format <#[TOKEN NAME]#>. For example, the string <#value#> adds a token labeled value to the snippet.

  4. Click Done.

Edit a custom code snippet

  1. Open a file in the source editor, then click the Snippets library button () in the toolbar.

  2. In the Snippets library, double-click the snippet, then click Edit.

  3. Edit the snippet properties and code.

    For a description of the properties, go to Create a custom code snippet.

  4. Click Done.

Delete a custom code snippet

  1. Open a file in the source editor, then click the Snippets library button () in the toolbar.

  2. Select the snippet you want to delete.

  3. Press Delete.
