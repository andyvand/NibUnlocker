Open a file in the hex editor

You can view and edit a file in its binary format by opening it in the hex editor.

  1. In the Project navigator (), Control-click the file, then choose Open As > Hex.

  2. From the Editor menu, toggle a column on and off:

    • Plain Text: Displays the plain-text (ASCII) representation of the bytes.

    • Line Offsets: Displays the line offsets.

    • Hexadecimal: Displays the hexadecimal representation of the bytes in the file, 32 bytes per line, one byte per column.

  3. From the Editor > Byte Grouping menu, choose a byte code group (2, 4, 8, 16, or 32).
